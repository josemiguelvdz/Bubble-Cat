Descripción

Se trata de un juego de plataformas con scroll lateral y donde encarnamos a Yuno, un gato con el poder de controlar el jabón.

El objetivo principal es acabar con nuestra vecina Bastet.

Para ello debemos adentrarnos en las alcantarillas y hacer uso de nuestros poderes para poder superarlas. Yuno tiene una pompa en la cabeza a modo de escudo, un pompero a la espalda con el que podremos hacer pompas y controlarlas, o pegar con él a corta distancia y un arma de pastillas de jabón para eliminar a los enemigos.

A lo largo de todo el juego nos encontraremos con puzzles y enemigos que requieren de usar nuestros poderes para avanzar.
